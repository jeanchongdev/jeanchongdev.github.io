Sobre mí

Soy Jean Pierre Chong, y empecé a programar a los 14 años, explorando el mundo del código con el Bloc de notas. Mi curiosidad inicial me llevó a aprender las bases de la lógica y a entender cómo pequeños fragmentos de código pueden resolver problemas de manera práctica.

Desde entonces, he desarrollado mis conocimientos en desarrollo web, enfocándome principalmente en HTML, CSS y JavaScript para crear interfaces interactivas y visualmente atractivas. Disfruto ver cómo el código puede transformar ideas en experiencias de usuario dinámicas y efectivas.
